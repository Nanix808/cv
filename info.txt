Клонировать репозиторий
-------------------------Настройка BACKEND----------------------
Установить python
Проверить что python виден из командной строки командой python --version,
иначе добавить python в переменную окружения
Перейти в папку cd backend
Создать виртуальное окружение
Активировать виртуальное окружение
В VS указать интепритатор виртуального окружения
Установить зависимости pip install -r requirements.txt
Убедиться что у вас на локальной машине создана 
база данных MySQL с такими же параметрами как в файле .env
Проверить сторонней программой подключение к базе данных
Создать миграцию
    alembic revision --autogenerate -m “First commit”
Применить миграцию
    alembic upgrade heads
Запустить файл main.py
Должен запуститься webserver на 127.0.0.1:8000
Перейти в браузере на этот адрес

-------------------------Настройка BACKEND----------------------
Установить nodejs https://nodejs.org/en
Проверить что nodejs виден из командной строки командой node -v,
иначе добавить nodejs в переменную окружения
Перейти в папку cd frontend
Запустить команду npm install
Запустить команду npm run dev, запустится webserver c каким-то ip и портом
Перейти в браузере на этот адрес
